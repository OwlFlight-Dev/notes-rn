# notes-rn

## Runtime Environment

- **React Native**: 0.72.x (managed with Expo)
- **Expo SDK**: 50.0.0
- **Node.js**: 18.x or higher
- **npm**: 9.x or higher
- **TypeScript**: 5.x
- **AsyncStorage**: `@react-native-async-storage/async-storage` latest
- **Expo Linear Gradient**: `expo-linear-gradient` latest
- **React Navigation**: 6.x (`@react-navigation/native`, `@react-navigation/native-stack`)

## Installation

```bash
# Clone the repo
git clone git@github.com:OwlFlight-Dev/notes-rn.git
cd notes-rn

# Install dependencies
npm install

# Start development server
npm start
